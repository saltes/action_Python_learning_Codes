## 快速开始

1. Fork此仓库
   
   首次使用时，你需要先**Fork此仓库**，然后再进行参数配置

2. 配置参数
  
   假设你此时已经Fork本仓库并进入自己的仓库，
   首先把你的代码放入Codes文件夹内
   点击菜单中 **Actions - 左栏All workflows下的右侧 Run workflow 灰色按钮
   - 填写相应参数**
   - 你的文件名例如
   - list列表常用操作课后练习.py
                                                            

3. 开始
   
   点击Run workflow绿色按钮即可开始
